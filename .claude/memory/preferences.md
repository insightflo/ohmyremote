# User Preferences

## 코드 스타일
- **스타일**: TypeScript 기본 스타일 준수
- **세미콜론**: 사용하지 않음
- **따옴표**: 작은따옴표 선호
- **들여쓰기**: 2 spaces (TypeScript)

## 명명 규칙
- **변수**: camelCase
- **상수**: UPPER_SNAKE_CASE
- **함수**: camelCase
- **인터페이스**: PascalCase
- **파일**: kebab-case.ts

## 프레임워크 선호
- **상태 관리**: 없음 (상태를 서버/API로 관리)
- **타입 검증**: Zod
- **ORM**: Drizzle ORM
- **라우터**: Fastify의 내장 라우터

## 패키지 매니저
- **pnpm** — workspace 사용

## 외부 라이브러리
- **grammY** — Telegram bot framework
- **Fastify** — API framework (v5)
- **Vite** — 웹 빌드
- **React** — 웹 UI
- **SQLite + Drizzle ORM** — 데이터베이스
- **Zod** — 스키마 검증
